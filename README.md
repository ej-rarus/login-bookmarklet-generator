# 🔐 로그인 북마크렛 생성기

> 테스트 계정으로 반복 로그인하기 귀찮아서 만든 자동 로그인 도구

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## ✨ 주요 기능

- 🎯 **자동 로그인 폼 감지** - HTML 파일만 업로드하면 자동으로 ID/비밀번호 필드 인식
- 👆 **시각적 요소 선택** - 자동 감지 실패 시 클릭만으로 직접 선택 가능
- 📊 **엑셀 업로드 지원** - 여러 계정을 엑셀로 한 번에 관리
- ✏️ **수동 입력** - 계정을 직접 입력하고 동적으로 추가/삭제
- 🔒 **100% 클라이언트 사이드** - 모든 처리가 브라우저에서만 이루어져 안전함

## 🚀 빠른 시작

1. **HTML 파일 준비**
   - 로그인 페이지에서 `Ctrl+S` (Mac: `Cmd+S`)로 저장

2. **파일 업로드**
   - index.html을 브라우저에서 열기
   - HTML 파일 업로드

3. **계정 정보 입력**
   - 엑셀 파일 업로드 또는 수동 입력

4. **북마크렛 생성 및 사용**
   - 생성된 코드를 브라우저 북마크에 저장
   - 로그인 페이지에서 북마크 클릭!

## 📖 사용 방법

자세한 사용법은 [튜토리얼 페이지](tutorial.html)를 참고하세요.

### 자동 감지 실패 시

"🎯 직접 선택하기" 기능을 사용하세요:
- ID 입력 필드를 클릭
- 비밀번호 입력 필드를 클릭
- 로그인 버튼을 클릭
- 완료!

## 🛠 기술 스택

- **순수 HTML/CSS/JavaScript** - 프레임워크 없이 가볍게
- **SheetJS (xlsx)** - 엑셀 파일 파싱
- **브라우저 API** - FileReader, DOMParser, iframe

## 🔒 보안

- **서버 전송 없음**: 모든 데이터는 브라우저 내에서만 처리
- **오프라인 작동**: 네트워크 연결 없이도 사용 가능
- ⚠️ **주의**: 북마크렛에는 계정 정보가 평문으로 저장되므로 **테스트 계정에만 사용**하세요

## 💡 왜 만들었나?

웹 개발 중 테스트 환경에서 여러 계정으로 반복 로그인하는 작업이 너무 귀찮았습니다.

**기존 방식**:
1. 아이디 입력
2. 비밀번호 입력
3. 로그인 버튼 클릭
4. 반복...

**북마크렛 사용**:
1. 북마크 클릭
2. 끝!
