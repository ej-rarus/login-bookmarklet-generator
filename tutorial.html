📖 사용 튜토리얼

북마크렛 생성기를 처음 사용하시나요? 단계별로 안내해드립니다!

🤔 북마크렛이란?

북마크렛(Bookmarklet)은 브라우저 북마크에 저장할 수 있는 작은 JavaScript 프로그램입니다. 북마크를 클릭하면 현재 페이지에서 특정 동작을 수행할 수 있습니다.

이 도구는 로그인 페이지를 분석하여 ID와 비밀번호를 자동으로 입력하고 로그인 버튼을 클릭하는 북마크렛을 생성해줍니다.

💡 언제 사용하나요?
테스트 환경에서 여러 계정으로 반복적으로 로그인해야 할 때 매우 유용합니다!

🚀 빠른 시작 가이드

STEP 1: 로그인 페이지 HTML 파일 준비

  1. 테스트할 로그인 페이지를 브라우저에서 엽니다
  2. Ctrl+S (Mac: Cmd+S) 눌러서 "다른 이름으로 저장" 실행
  3. 파일 형식을 "웹페이지, 완성본" 또는 "HTML 전용"으로 선택
  4. 원하는 위치에 저장하기
💡 정말 쉽죠?
그냥 로그인 페이지에서 Ctrl+S만 누르면 끝! 복사/붙여넣기 필요 없어요.

STEP 2: HTML 파일 업로드

  1. 북마크렛 생성기로 돌아갑니다
  2. "1️⃣ 로그인 페이지 HTML 파일" 영역에 파일을 드래그하거나 클릭하여 업로드
  3. 자동으로 로그인 폼 요소를 감지합니다
💡 자동 감지 실패 시
미리보기에서 "🎯 직접 선택하기" 버튼을 클릭하여 ID 필드, 비밀번호 필드, 로그인 버튼을 직접 선택할 수 있습니다!

STEP 3: 계정 목록 입력

두 가지 방법 중 선택:

방법 1: 엑셀 파일 업로드
  • 엑셀 파일에 ID, Password 컬럼 생성
  • 테스트 계정 정보 입력
  • "2️⃣ 테스트 계정 목록"에서 파일 업로드
방법 2: 직접 입력
  • 테이블에서 계정 정보를 직접 입력
  • "+ 계정 추가" 버튼으로 행 추가
  • 필요 없는 행은 "삭제" 버튼으로 제거

STEP 4: 북마크렛 생성

  1. "북마크렛 생성하기" 버튼 클릭
  2. 목록형 표로 북마크렛들이 생성됨

STEP 5: 북마크 등록 (3가지 방법 중 선택)

방법 1: 드래그 앤 드롭 (추천!) 🖱️
  • 표에서 📌 계정 이름 링크를 찾기
  • 링크를 브라우저 북마크바로 드래그
  • 끝! 가장 쉬운 방법입니다
방법 2: HTML 파일로 일괄 등록 📥
  • "📥 모든 북마크 HTML로 내보내기" 버튼 클릭
  • login-bookmarklets.html 파일 다운로드
  • 브라우저 설정 → 북마크 → 북마크 가져오기
  • 다운로드한 파일 선택
  • 모든 북마크가 한 번에 등록됩니다!
방법 3: 코드 복사하여 수동 등록 📋
  • "📋 코드 복사" 버튼 클릭
  • 브라우저 북마크바에서 우클릭 → "새 북마크 추가"
  • 이름: 계정 이름 입력
  • URL: 복사한 코드 붙여넣기
  • 저장

STEP 6: 사용하기

  1. 로그인 페이지로 이동
  2. 등록한 북마크를 클릭
  3. 자동으로 ID/비밀번호가 입력되고 로그인됩니다! 🎉

📌 북마크 등록 방법 상세 가이드

🖱️ 방법 1: 드래그 앤 드롭 (가장 추천!)

가장 쉽고 빠른 방법입니다:

  1. 북마크렛 생성 후 표에서 📌 계정 이름 링크를 찾습니다
  2. 링크를 클릭한 상태로 드래그합니다
  3. 브라우저 북마크바로 드래그하여 놓습니다
  4. 북마크가 자동으로 생성됩니다!
💡 Tip
브라우저 북마크바가 안 보이면 Ctrl+Shift+B (Mac: Cmd+Shift+B)로 표시/숨기기 가능

📥 방법 2: HTML 파일로 일괄 등록

여러 북마크를 한 번에 등록할 때 편리합니다:

  1. 북마크렛 생성 후 "📥 모든 북마크 HTML로 내보내기" 버튼 클릭
  2. login-bookmarklets.html 파일이 자동으로 다운로드됩니다
  3. 브라우저 설정 열기 (Chrome: ⋮ → 북마크 → 북마크 관리자)
  4. 우측 상단 ⋮ → 북마크 가져오기 선택
  5. 다운로드한 HTML 파일 선택
  6. "로그인 북마크렛" 폴더에 모든 북마크가 추가됩니다!

📋 방법 3: 코드 복사하여 수동 등록

전통적인 방법입니다:

  1. 북마크렛 표에서 "📋 코드 복사" 버튼 클릭
  2. 클립보드에 코드가 복사됩니다
  3. 브라우저 북마크바에서 우클릭 → "새 북마크 추가" 선택
  4. 이름: 원하는 이름 입력 (예: "테스트 계정1")
  5. URL: 복사한 코드 붙여넣기 (Ctrl+V)
  6. 저장 버튼 클릭

🎯 직접 선택하기 기능 사용법

자동 감지가 실패했을 때 사용하는 기능입니다. 코딩 지식이 없어도 눈으로 보고 클릭만 하면 됩니다!

사용 방법

  1. HTML 파일 업로드 후 미리보기가 표시됩니다
  2. "🎯 직접 선택하기" 버튼 클릭
  3. 안내에 따라 순서대로 클릭:
    • 1단계: ID 입력 필드 클릭 (초록색으로 하이라이트됨)
    • 2단계: 비밀번호 입력 필드 클릭
    • 3단계: 로그인 버튼 클릭
  4. 잘못 선택했다면 "↶ 이전 단계" 버튼으로 되돌리기
  5. 완료되면 자동으로 설정이 저장됩니다!

❓ 자주 묻는 질문 (FAQ)

Q1. 북마크렛이 작동하지 않아요

다음 사항을 확인해보세요:

  • 드래그 앤 드롭으로 등록했다면 올바르게 등록되었는지 확인 (북마크 편집에서 URL 확인)
  • 코드 복사 방식이라면 전체 코드를 정확히 복사했는지 확인
  • 로그인 페이지의 구조가 변경되지 않았나요?
  • 브라우저 콘솔(F12)에서 에러 메시지를 확인해보세요

Q2. 자동 감지가 계속 실패해요

"🎯 직접 선택하기" 기능을 사용하세요. 로그인 폼의 구조가 특이한 경우 자동 감지가 어려울 수 있습니다.

Q3. 엑셀 파일 형식은 어떻게 되나요?

다음과 같이 구성하세요:

  • 1행: 헤더 (ID, Password, 이름)
  • 2행부터: 실제 계정 정보
  • 컬럼명은 대소문자 구분 없음 (id, ID, user 등 모두 가능)

Q4. 보안은 괜찮나요?

이 도구는 100% 클라이언트 사이드에서 동작합니다. 파일이나 계정 정보가 서버로 전송되지 않으며, 모든 처리는 여러분의 브라우저에서만 이루어집니다.

⚠️ 주의사항
북마크렛에는 계정 정보가 평문으로 저장됩니다. 실제 프로덕션 계정이 아닌 테스트 계정에만 사용하세요!
